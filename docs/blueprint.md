# **App Name**: GetItWebbed

## Core Features:

- Cinematic Portfolio Showcases: An interactive project gallery with image zooming, custom overlay reveals, and masonry layouts.
- AI Project Scoping Tool: An AI-powered consultation tool that uses reasoning to analyze user project descriptions and recommend appropriate tech stacks and features.
- Multi-Step Enquiry System: Comprehensive project inquiry form with validation, multi-select service options, and budget categorization.
- Real-time Enquiry Storage: Secure storage of all potential lead data and project briefs in a Firestore database.
- Animated Process Timeline: A horizontal scrolling progress tracker that illustrates the discovery-to-launch agency methodology.
- Interactive IoT Network Visualization: A Three.js or particle canvas background for the hero section representing connected hardware nodes.
- Tech Stack Marquee: Infinite looping horizontal ticker showing industry-standard tools and frameworks for Web and Mobile development.

## Style Guidelines:

- Primary color: Electric Mint (#00FFB2) used for CTAs and interactive highlights, offering high contrast against the dark scheme.
- Background color: Deep Space Black (#0A0A0A), desaturated and moody to emphasize cinematic content.
- Accent color: Tech Purple (#7B61FF) used sparingly for IoT highlights and gradients to signify innovation.
- Headline font: 'Space Grotesk' (sans-serif) for a bold, scientific, and technical aesthetic. Body font: 'Inter' (sans-serif) for objective, clean readability. Code snippets use 'Source Code Pro' (monospace).
- Full-bleed sections with generous white space (or dark space) and oversized decorative section numbers (01, 02) to maintain an editorial, agency feel.
- Advanced interactions using GSAP for staggered clip-path text reveals, grain texture overlays, and smooth scrolling via Lenis.
- Custom SVG iconography representing Web, App, and IoT sectors with subtle glowing hover states in the primary mint hue.
